# mini-project-guess-the-number
A simple number guessing game built using HTML, CSS and JavaScript

---

## Description
This is a small interative we game biult using **HTML, CSS, & JavaScript**.
The system generates a random number between **1 and 10**, and the user has to guess it.
Based on the input, the game provides instant feedback.

---
## Problem Statement
Build a basic web-based game to practice:
- JavaScript logic
- DOM manipulation
- Handling user input and events
  
---

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
1. The system generates a random number between 1 and 10.
2. The user enters a number in the input field.
3. On clicking the **Check** button:
   - If the guess is correct → success message is shown.
   - If the guess is higher or lower → feedback is displayed accordingly.

---

##  How to Run
1. Clone the repository
2. Open `index.html` in any web browser
3. Start guessing 

No additional setup required.
